背稿实战方法

三段式记忆法

第1-2天：关键词锚定

每页提取1-3个关键词写在A4纸上，按顺序默念。关键词链示例：
Slogan → 政策东风 → 三大痛点 → 120亿 → 三层架构 → 仿生骨架 → 95% → 竞争有利 → 3980元 → 四学科 → 曾艺文 → 1+1+8 → 四级盈利 → 三阶段 → 47倍 → 90万10%

第3-4天：段落串联

将每页拓展为2-4句完整台词，对照PPT逐页翻页练习，熟悉页间过渡逻辑。

第5-7天：全篇脱稿

不看PPT和讲稿，只凭关键词提示完整讲述。录视频自查口头禅、停顿、眼神。

数字记忆技巧——五组口诀

口诀含义
一二零·四五·二八TAM 120亿 → SAM 45亿 → SOM 2800万
三九八零·三九八硬件3980元/台 → 耗材398元/套
九五·三七·五百准确率95% → 提升37% → 500+组数据
九零·一零·五二融资90万 → 出让10% → IRR 52%
六八·三二·四七68万 → 3200万 → 47倍增长

镜像练习法

对镜子练习，观察手势是否自然、表情是否僵硬；用手机录像自查，标记口头禅（如"然后""就是说"）、停顿过长、眼神飘忽等问题点；邀请队友旁听模拟评委，获取第三人反馈。

时间控制训练

每页用秒表逐页计时，标注超时页；超时砍词而非加速语速——加速会让评委听不清；至少完成3次完整全流程；目标总时间稳定在7分30秒左右，不早于7分钟、不晚于8分钟。

答辩黄金法则

先结论后解释第一句直接给答案，再展开1-2句解释，控制在40秒内。
数据说话不用"很多""非常""极大"等形容词，必须说具体数字。
不会就说在探索真诚比硬撑更重要——"这个问题我们目前还在探索中，目前的思路是……"
不抢话不打断评委说完再回答，如果没听清可以礼貌请评委重复。
控制回答长度每个回答不超过40秒。4分钟答辩约能回答6-8个问题。

最可靠、零技术风险。即使电脑死机，手机照样计时。建议作为所有方案的兜底。

Q1 你们的核心技术壁垒是什么？
A 我们有三大壁垒：第一，自研柔性薄膜压力传感器阵列，64点矩阵采集，0.1N精度，市面上没有同类TCM专用传感器；第二，DTW动态时间规整算法引擎，将穴位按摩动作数字化匹配，经2000+临床样本验证准确率95%；第三，361个穴位×5种标准手法的专有数据库，这是3年以上中医临床专家标注的成果。三项发明专利已提交，构成完整的知识产权护城河。
Q2 你们的竞品有哪些？核心竞争力是什么？
A 目前市场上没有直接竞品。间接竞品有三类：传统中医训练人偶——没有数字反馈，靠师傅肉眼判断；通用医学模拟系统如CAE Healthcare——主要面向西医急救，没有中医穴位体系；可穿戴IMU手套如Rokoko——只有姿态捕捉，没有压力感应。我们的核心竞争力是"传感器+算法+数据库"三位一体，专为中医穴位定量评估而设计，这是其他产品不具备的。
Q3 商业模式是什么？怎么盈利？
A 我们有三级收入模型。第一级，硬件销售：智能传感手套+软件授权，定价3980元/套，面向中医高校和教学医院批量采购；第二级，SaaS订阅：云端数据分析+穴位数据库更新，1200元/学生/年；第三级，定制化数据库授权：为考培机构提供专项穴位考核数据库。收入结构预期：硬件60%、SaaS 25%、数据库授权15%。
Q4 融资90万怎么花？估值怎么来的？
A 90万的使用分配：40%用于硬件研发优化，主要是传感器模具开模和柔性材料迭代；30%用于临床验证，在3家教学医院开展300例对照试验；20%用于软件开发，完成云端分析平台MVP；10%用于知识产权和医疗器械注册备案。融资前估值810万，对应出让10%股权。估值基于：已投入研发成本约120万、3项发明专利的IP价值、以及已有2家意向采购单位的市场验证。
Q5 目标市场有多大？你们瞄准哪个细分？
A 中医教育市场总规模TAM约1200亿元，可服务市场SAM约180亿——涵盖30+所中医高等院校、2000+家中医教学医院、以及全国中医执业医师技能考核市场。我们头三年瞄准60家试点院校和医院，目标市场SOM约3.6亿。中长期扩展至中医药管理局技能认证体系、海外中医培训机构，市场空间非常可观。
Q6 DTW算法具体怎么工作？为什么用它？
A DTW（动态时间规整）是一种专门处理时间序列相似度匹配的算法。学生按摩穴位时，传感器以100Hz采样记录力度、角度、深度三个维度的时间序列数据，形成一个多维轨迹曲线。DTW将这条曲线与标准手法模板进行非线性对齐和距离计算，输出0-100分的匹配评分。选DTW而非传统深度学习的原因：第一，中医手法速度快慢因人而异，DTW天然支持时间轴伸缩；第二，训练数据有限，DTW不需要大量标注样本，基于模板即可工作；第三，可解释性强，能指出具体哪一步偏离了标准，方便教学反馈。
Q7 团队怎么构成的？跨学科协作如何保障？
A 团队共7人，四学科交叉：中医临床2人——负责穴位数据库建设、手法标准化和临床验证设计；硬件工程1人——负责柔性传感器选型、电路设计和嵌入式开发；算法软件2人——负责DTW引擎、云端平台和数据可视化；商业运营2人——负责市场渠道、融资和医疗器械注册。跨学科协作有制度化保障：每周联合评审会、共享GitLab文档库、以及一位从业20年的副主任中医师担任临床顾问把关方向，确保工程实现不走偏中医本质。
Q8 产品目前到了什么阶段？
A 目前是原型机V2.0阶段，已经完成：柔性传感器手套硬件迭代两版、IMU姿态模块集成、361穴位标准手法数据库建库、DTW评分引擎完成2000例测试（准确率95%）、3项发明专利已提交、1项软件著作权已登记。下一阶段重点是：与3家医院合作开展300例临床验证试验，同步开发云端分析平台MVP，并启动二类医疗器械注册准备。预计6个月内完成临床验证并进入小批量试产。
Q9 面对的最大风险是什么？如何应对？
A 我们识别了三大风险并都有应对方案。技术风险：柔性传感器长期耐用性——我们已与中科院深圳先进院材料所建立合作，测试目标寿命10万次按压，目前通过5万次。市场风险：中医教育数字化接受度——我们采用"试点先行"策略，已获得2家院校的口头采购意向，用标杆案例打消疑虑。监管风险：医疗器械认证周期长——我们先行以教学设备（非医疗器械）切入市场，同步启动二类医疗器械注册流程，双轨并行不会卡住商业化进度。
Q10 未来三年规划是什么？
A 三年三阶段：第一年（验证期）——完成300例临床验证，拿到二类医疗器械备案，签约10家院校客户，营收240万；第二年（扩展期）——产品迭代V3.0，增加针灸手法评估模块，扩展至50家客户，SaaS收入占比提升至30%，营收850万；第三年（规模化）——进入中医执业资格考试市场，与中医药管理局合作技能认证标准，客户数突破150家，营收2200万。三年后启动Pre-A轮融资，估值目标5000万。
Q11 传感器用的是哪种技术？有什么优势？
A 我们使用的是柔性薄膜压阻式传感器阵列，64个采集点分布在手套指尖和掌面，量程0-50N，分辨率0.1N。相比市面上主流的电容式传感器，压阻式优势在于：第一，线性度好，力-电转换关系稳定，不需要复杂校准；第二，抗干扰强，不受汗液和皮肤湿度影响，这是中医按摩场景的刚需；第三，成本可控，单只手套传感器模组成本约180元，量产可降至80元以内，支撑3980元定价的健康毛利。
